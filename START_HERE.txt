## 🎯 What You Need To Do

### ✅ STEP 1: Gather Data (CRITICAL - Takes Weeks)

**Requirements:**
1. **500-1000+ X-ray screw images** per screw type (M3, M4, M5, M6)
2. **3D CAD models** of screws in PLY format (millimeter scale)
   - Get from supplier CAD, or generate from ISO standard specs
   - Must include all 8 corners visible in 3D mesh
3. **Camera calibration parameters** (fx, fy, u0, v0)
   - From X-ray detector specs, OR
   - Use OpenCV charuco board calibration

**Action:**
```bash
# Copy images to appropriate directories
# For M3 screws:
copy your_xray_images/*.jpg → data/XRAY_SCREWS/m3/JPEGImages/
copy m3_model.ply → data/XRAY_SCREWS/m3/m3.ply
# (repeat for m4, m5, m6)
```

---

### ✅ STEP 2: Label Your Images

Use the **interactive annotation tool** to click 9 keypoints per image:
- Keypoint 0: Center of screw head
- Keypoints 1-8: Eight corners of screw bounding box

**Commands:**
```bash
# Label M3 images
cd xray-screw-detection
python data_curation/annotate_screws.py \
  --images data/XRAY_SCREWS/m3/JPEGImages \
  --output data/XRAY_SCREWS/m3/Annotations \
  --class-id 0

# Label M4 images (different --class-id):
python data_curation/annotate_screws.py \
  --images data/XRAY_SCREWS/m4/JPEGImages \
  --output data/XRAY_SCREWS/m4/Annotations \
  --class-id 1

# (repeat for m5 with --class-id 2, m6 with --class-id 3)
```

**Keyboard Controls in Annotation Tool:**
- **LEFT-CLICK:** Add keypoint (1-9 total)
- **S:** Save annotation & go to next image
- **D:** Delete last point
- **R:** Reset all points for current image
- **N:** Load next image
- **P:** Load previous image
- **Q:** Quit tool

**💡 Tip:** The tool saves visualization images with `_vis.jpg` suffix so you can review your work.

### ✅ STEP 5: Configure Your System (CRITICAL!)

**Edit:** `configs/screws/screw_camera.json`

```json
{
  "camera_matrix": [
    [YOUR_FX, 0.0, YOUR_U0],
    [0.0, YOUR_FY, YOUR_V0],
    [0.0, 0.0, 1.0]
  ],
  ...
}
```

**UPDATE these values with YOUR X-ray detector calibration:**
- `fx`: Focal length in X direction (pixels)
- `fy`: Focal length in Y direction (pixels)  
- `u0`: Principal point X (usually image_width / 2)
- `v0`: Principal point Y (usually image_height / 2)

**How to get these:**
1. **From detector specs:** Manufacturer usually provides calibration file
2. **Calibrate yourself:** Use OpenCV + charuco board
   ```python
   # See: https://docs.opencv.org/master/d5/dae/tutorial_aruco_calibration.html
   ```

⚠️ **ACCURACY OF THESE VALUES DIRECTLY AFFECTS POSE ESTIMATION ACCURACY!**

### ✅ STEP 7: Train the Model

Choose model size based on your GPU:

```bash
# Small & Fast (4GB GPU):
python train.py \
  --batch 32 \
  --epochs 300 \
  --cfg models/yolov5s_screw_bifpn.yaml \
  --hyp configs/hyp.single.yaml \
  --weights yolov5s.pt \
  --data configs/screws/m3.yaml \
  --img 640 \
  --rect \
  --cache \
  --device 0

# OR Medium (8GB GPU):
python train.py \
  --cfg models/yolov5m_screw_bifpn.yaml \
  --weights yolov5m.pt \
  ... (same other args)

# OR Large & Accurate (12GB+ GPU):
python train.py \
  --cfg models/yolov5l_screw_bifpn.yaml \
  --weights yolov5l.pt \
  ... (same other args)
```

**Monitor Training (Real-Time):**
```bash

```

## 🐛 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| "No module named 'torch'" | Run: `pip install -r requirements.txt` |
| Annotation tool hangs | Make sure images are in JPEGImages/ folder |
| Dataset validation fails | Run again, check output for specific errors |
| Training loss = NaN | Reduce lr0 in configs/hyp.single.yaml (0.0001) |
| CUDA out of memory | Reduce batch size (--batch 8) or use smaller model |
| Low accuracy (ADD-S > 5mm) | 1) Check camera calibration 2) More data 3) More epochs |

## 🎓 Annotation Format (Reference)

Each `.txt` file must have exactly **21 values**:

```
<class_id> <x0> <y0> <x1> <y1> <x2> <y2> ... <x8> <y8> 1.0 1.0
│            └──────────────────────────────────────────X─────┘
│            9 keypoints (18 coordinates), normalized to [0,1]
└─ class: 0 for m3, 1 for m4, etc.
```

**Keypoint layout:**
```
8 corners of screw head bounding box:
      7 ─────────────── 6
     /│              /│
    5 ─────────────── 4 │
    │ 3 ────────────│── 2
    │/              │ /
    1 ────────────── 0
    
Plus point 0 = center of head
```
