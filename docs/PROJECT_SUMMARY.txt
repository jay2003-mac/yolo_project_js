🚀 QUICK START GUIDE
================================================================================

1. READ FIRST:
   Open: START_HERE.txt
   (Located in xray-screw-detection folder)

2. GATHER DATA:
   - Collect 500+ X-ray screw images per type
   - Obtain 3D screw CAD models (PLY format, mm scale)
   - Get camera calibration parameters (fx, fy, u0, v0)

3. PLACE YOUR DATA:
   - Images  → data/XRAY_SCREWS/m3/JPEGImages/
   - 3D mesh → data/XRAY_SCREWS/m3/m3.ply
   - (similar for m4, m5, m6)

4. LABEL YOUR IMAGES:
   python data_curation/annotate_screws.py \
     --images data/XRAY_SCREWS/m3/JPEGImages \
     --output data/XRAY_SCREWS/m3/Annotations

5. CONFIGURE SYSTEM:
   Edit: configs/screws/screw_camera.json
   Add your X-ray detector calibration values

6. VALIDATE DATA:
   python data_curation/validate_dataset.py --dataset data/XRAY_SCREWS/m3

7. TRAIN THE MODEL:
   python train.py \
     --data configs/screws/m3.yaml \
     --batch 32 \
     --epochs 300 \
     --device 0

⚠️ CRITICAL FILES YOU MUST CUSTOMIZE
================================================================================

1. Camera Intrinsics (MOST IMPORTANT!)
   File: configs/screws/screw_camera.json
   What: YOUR X-ray detector's focal length & principal point
   Action: Get from manufacturer specs or calibrate with charuco board
   
2. Training Image Paths
   File: data/XRAY_SCREWS/m3/train.txt
   What: List of training image files (70% of your data)
   Action: One path per line, e.g., ../data/XRAY_SCREWS/m3/JPEGImages/img_001.jpg
   
3. Test Image Paths
   File: data/XRAY_SCREWS/m3/test.txt
   What: List of validation image files (30% of your data)
   Action: One path per line

4. 3D Screw Model
   File: data/XRAY_SCREWS/m3/m3.ply
   What: 3D CAD mesh of screw (MUST be in millimeters!)
   Action: Place CAD file here
